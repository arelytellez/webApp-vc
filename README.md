# WebApp - Reconocimiento de Gestos

Aplicación web desarrollada con MediaPipe y Bootstrap para identificar órdenes mediante gestos de la mano utilizando la cámara web.

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- MediaPipe

## Funcionalidades
- Reconocimiento de gestos
- Modo suspendido automático
- Activación por movimiento
- Mensaje de orden no reconocida
